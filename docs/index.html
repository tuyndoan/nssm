Run anything as a Windows Service

NSSM wraps any application as a Windows Service and restarts it when it exits — like srvany / cygrunsrv, but without the suck. GUI installer + full CLI.

Quick start

Wrap an app in seconds, or build from source

Manage a service
:: install & start
nssm install MyApp "C:\app\app.exe" --flag
nssm start   MyApp

:: edit in GUI / tweak & remove
nssm edit    MyApp
nssm remove  MyApp confirm
Build from source
:: 32-bit (default) / 64-bit
build.cmd
build.cmd x64

:: package win32 + win64 → dist/
pack.cmd

:: quick check
build\out\x86\nssm.exe version
